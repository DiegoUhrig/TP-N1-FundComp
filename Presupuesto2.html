4. Justificación de Componentes

🔥 AMD Ryzen 5 5600 - Procesador

Características técnicas: 6 núcleos y 12 hilos con arquitectura Zen 3, frecuencia base de 3.5 GHz y boost de hasta 4.4 GHz. Excelente rendimiento single-core y multi-core.

Por qué se eligió: Balance perfecto para gaming y streaming simultáneo. Los 12 hilos permiten dedicar núcleos al juego y otros al encoding de OBS. Mejor precio/rendimiento en el rango de $150-200.

🎮 NVIDIA GeForce RTX 3060 - Tarjeta Gráfica

Características técnicas: 12GB GDDR6, 3584 CUDA cores, soporte para Ray Tracing y DLSS, encoder NVENC de tercera generación.

Por qué se eligió: El encoder NVENC permite streaming de alta calidad sin impacto en el rendimiento del juego. Los 12GB de VRAM son ideales para Minecraft con mods y texturas HD. Soporta 1080p @ 60fps con margen para futuras actualizaciones.

💾 16GB DDR4-3200 - Memoria RAM

Características técnicas: Dual channel 2x8GB, velocidad 3200MHz, latencia CL16, compatible con perfiles XMP.

Por qué se eligió: Mínimo requerido para streaming + gaming. Minecraft puede usar 6-8GB con mods, OBS necesita 4GB, y el sistema 4GB adicionales.

⚡ Kingston NV2 500GB M.2 - Almacenamiento

Características técnicas: SSD NVMe PCIe 4.0, velocidades de hasta 3500MB/s en lectura.

Por qué se eligió: Los tiempos de carga rápidos son cruciales para streaming (OBS, juego, navegador simultáneamente). 500GB suficientes para sistema, Minecraft, OBS y archivos de grabación básicos.

  • FPS con shaders: 60-90 FPS @ 1080p (BSL, Complementary)

🔴 Capacidades de Streaming:

  • 1080p @ 60fps: Excelente calidad con bitrate 6000 Kbps

  • Encoding: NVENC para máximo rendimiento, x264 disponible

8. Configuración de Software Recomendada

🎥 OBS Studio - Configuración Optimizada:

  • Encoder: NVENC (nuevo) para RTX 3060
  • Rate Control: CBR (Constant Bitrate)
  • Bitrate: 6000 Kbps para 1080p60 / 4500 Kbps para 720p60
  • Preset: "Quality" para máxima calidad de imagen
  • Profile: High
  • Look-ahead: Activado
  • Psycho Visual Tuning: Activado

⚙️ Minecraft - Configuración para Streaming:

  • Render Distance: 12-16 chunks (balance rendimiento/visual)
  • VSync: Desactivado (usar limitador de FPS a 120)
  • Max FPS: 120 FPS (deja headroom para encoding)
  • RAM asignada: 6-8GB (-Xmx8G en launcher)
  • OptiFine/Sodium: Recomendado para optimización adicional
